# Invoice Clipper 

## How it works
- Provide one input: an extraction prompt in the sidepanel.
- Upload one or more PDF invoices.
- Extension sends `prompt + PDF` to Gemini.
- Results can be copied per field and exported to CSV.

## Setup
1. Load this folder as an unpacked extension in Chrome.
2. Get a gemini API key at https://aistudio.google.com/app/apikey.
3. Set your gemini key in the settings.
